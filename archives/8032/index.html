gitеҹәжң¬ж“ҚдҪңпјҢдёҖзҜҮж–Үз« е°ұеӨҹдәҶ

|
2021-10-10
| 1.1kеӯ— | 4еҲҶй’ҹ
git

еёёиҜҶеҢә

  git add . // жӣҙж–°еҲ°жҡӮеӯҳеҢә
  git commit -m "feat: ж·»еҠ  Hello.ts йЎөйқў" // жӣҙж–°еҲ°иө„жәҗеә“
    // жҸҗдәӨи§„иҢғ:
    1. feat: еўһеҠ ж–°еҠҹиғҪ
    2. fix: дҝ®еӨҚbug
    3. docs: еҸӘж”№еҠЁдәҶж–ҮжЎЈзӣёе…іеҶ…е®№
    4. styles: дёҚеҪұе“Қд»Јз Ғзҡ„еҗ«д№үзҡ„ж”№еҠЁпјҢеҰӮпјҡеҺ»жҺүз©әж јгҖҒзј©иҝӣгҖҒеўһеҲ еҲҶеҸ·зӯү
    5. build: жһ„йҖ е·Ҙе…·жҲ–иҖ…еӨ–йғЁдҫқиө–зҡ„ж”№еҠЁ еҰӮwebpackгҖҒnpmзӯү
    6: refactor: д»Јз ҒйҮҚжһ„ж—¶дҪңз”Ё
  git commit --amend // vimдёӯдҝ®ж”№еҶ…е®№дҝЎжҒҜ
  git push // жҸҗдәӨиҮіиҝңзЁӢд»“еә“

  git branch -D <BranchName> // еҲ йҷӨжң¬ең°еҲҶж”Ҝ
    git push origin --delete <BranchName>  // зңҹжӯЈзҡ„еҲ йҷӨиҝңзЁӢеҲҶж”Ҝ

йҮҚзӮ№еҢә

  [git reset & git revet]

  1. gitеҰӮдҪ•ж’Өй”ҖдёҠдёҖж¬Ўcommitж“ҚдҪң ???

  // жңҖеҗҺдёҖж¬ЎжҸҗдәӨзҡ„дҝ®ж”№пјҢеҢ…жӢ¬жң¬ең°ж–Үд»¶зҡ„дҝ®ж”№йғҪдјҡиў«жё…йҷӨпјҢеҪ»еә•иҝҳеҺҹеҲ°дёҠдёҖж¬ЎжҸҗдәӨзҡ„зҠ¶жҖҒдё”ж— жі•жүҫеӣһгҖӮ
  // жүҖд»ҘеңЁжү§иЎҢreset --hardд№ӢеүҚдёҖе®ҡиҰҒе°Ҹеҝғ
  `git reset --hard HEAD~` жҲ–иҖ…жҢҮе®ҡеӣһйҖҖжҸҗдәӨзҡ„commidHash `git reset --hard <commitHash>`

  // йҮҚзҪ®HEADеҲ°жҢҮе®ҡзҡ„зүҲжң¬, дёҚдјҡдҝ®ж”№indexе’Ңworking tree
  `git reset --soft <commitHash>`


    git reset (--mixed) HEAD~1В 
    еӣһйҖҖдёҖдёӘзүҲжң¬,дё”дјҡе°ҶжҡӮеӯҳеҢәзҡ„еҶ…е®№е’Ңжң¬ең°е·ІжҸҗдәӨзҡ„еҶ…е®№е…ЁйғЁжҒўеӨҚеҲ°жңӘжҡӮеӯҳзҡ„зҠ¶жҖҒ,дёҚеҪұе“ҚеҺҹжқҘжң¬ең°ж–Үд»¶(жңӘжҸҗдәӨзҡ„д№ҹВ дёҚеҸ—еҪұе“Қ)В еӣһйҖҖеҗҺзҡ„д»Јз ҒпјҲе·ІжҸҗдәӨзҡ„еҶ…е®№пјүж”ҫеңЁе·ҘдҪңеҢәпјҢе·ҘдҪңеҢәе’ҢжҡӮеӯҳеҢәзҡ„дҝ®ж”№йғҪдјҡз•ҷдёӢ

    git reset --soft HEAD~1В 
    еӣһйҖҖдёҖдёӘзүҲжң¬,дёҚжё…з©әжҡӮеӯҳеҢә,е°Ҷе·ІжҸҗдәӨзҡ„еҶ…е®№жҒўеӨҚеҲ°жҡӮеӯҳеҢә,дёҚеҪұе“ҚеҺҹжқҘжң¬ең°зҡ„ж–Үд»¶(жңӘжҸҗдәӨзҡ„д№ҹдёҚеҸ—еҪұе“Қ)В 
    е°Ҷе·ІжҸҗдәӨзҡ„еҶ…е®№жҒўеӨҚеҲ°жҡӮеӯҳеҢәпјҢеҺҹжқҘжҡӮеӯҳеҢәе’Ңе·ҘдҪңеҢәзҡ„еҶ…е®№д»Қз„¶дҝқз•ҷ

    git reset --hard HEAD~1В 
    еӣһйҖҖдёҖдёӘзүҲжң¬,жё…з©әжҡӮеӯҳеҢә,е°Ҷе·ІжҸҗдәӨзҡ„еҶ…е®№зҡ„зүҲжң¬жҒўеӨҚеҲ°жң¬ең°,жң¬ең°зҡ„ж–Үд»¶д№ҹе°Ҷиў«жҒўеӨҚзҡ„зүҲжң¬жӣҝжҚў
    е·ІжҸҗдәӨзҡ„еҶ…е®№жҒўеӨҚеҲ°е·ҘдҪңеҢәпјҢеҺҹжқҘе·ҘдҪңеҢәдёӯзҡ„дҝ®ж”№дјҡз•ҷдёӢпјҢеҺҹжқҘжҡӮеӯҳеҢәдёӯзҡ„еҶ…е®№дјҡж¶ҲеӨұ

  --mixed  дјҡдҝқз•ҷжәҗз Ғ,еҸӘжҳҜе°Ҷgit commitе’Ңindex дҝЎжҒҜеӣһйҖҖеҲ°дәҶжҹҗдёӘзүҲжң¬.
  --soft   дҝқз•ҷжәҗз Ғ,еҸӘеӣһйҖҖеҲ°commitдҝЎжҒҜеҲ°жҹҗдёӘзүҲжң¬.дёҚж¶үеҸҠindexзҡ„еӣһйҖҖ,еҰӮжһңиҝҳйңҖиҰҒжҸҗдәӨ,зӣҙжҺҘcommitеҚіеҸҜ.
  --hard   жәҗз Ғд№ҹдјҡеӣһйҖҖеҲ°жҹҗдёӘзүҲжң¬,commitе’Ңindex йғҪдјҡеӣһйҖҖеҲ°жҹҗдёӘзүҲжң¬.(жіЁж„Ҹ,иҝҷз§Қж–№ејҸжҳҜж”№еҸҳжң¬ең°д»Јз Ғд»“еә“жәҗз Ғ)

  2. commit push д»Јз Ғе·Із»Ҹжӣҙж–°еҲ°иҝңзЁӢд»“еә“ еҰӮдҪ•ж’Өй”Җ пјҹпјҹпјҹ

  // revert д№ӢеҗҺдҪ зҡ„жң¬ең°д»Јз Ғдјҡеӣһж»ҡеҲ°жҢҮе®ҡзҡ„еҺҶеҸІзүҲжң¬,иҝҷж—¶дҪ еҶҚ git push ж—ўеҸҜд»ҘжҠҠзәҝдёҠзҡ„д»Јз Ғжӣҙж–°
  `git revert <commit_id>`
  // жіЁж„Ҹпјҡgit revertжҳҜз”ЁдёҖж¬Ўж–°зҡ„commitжқҘеӣһж»ҡд№ӢеүҚзҡ„commitпјҢgit resetжҳҜзӣҙжҺҘеҲ йҷӨжҢҮе®ҡзҡ„commitпјҢзңӢдјјиҫҫеҲ°зҡ„ж•ҲжһңжҳҜдёҖж ·зҡ„,е…¶е®һе®Ңе…ЁдёҚеҗҢгҖӮ

  [git stash]

  // е…ЁйғЁжңӘдҝқеӯҳзҡ„д»Јз Ғж·»еҠ еҲ°еӮЁи—Ҹ
  git stash жҲ– git stash save "message"
    1. git stash pop // еә”з”Ёз¬¬дёҖдёӘеӮЁи—Ҹе№¶еҲ йҷӨпјҢеҰӮжһңpopеҜ№еә”йЎ№пјҢеҲҷзӣҙжҺҘpopеҗҺйқўи·ҹеҜ№еә”зҡ„еәҸеҲ—еҸ·е°ұеҸҜд»Ҙ
    2. git stash apply // еә”з”Ёз¬¬дёҖдёӘеӮЁи—ҸдҪҶдҫқз„¶дҝқз•ҷеңЁеҲ—иЎЁйЎ№дёӯгҖӮ
    3. git stash list // жҹҘзңӢstashеҲ—иЎЁ
    4. git stash drop stash@{xxx} // еҲ йҷӨжҹҗйЎ№еӮЁи—Ҹ
    5. git stash clear // еҲ йҷӨжүҖжңүеӮЁи—Ҹзҡ„stash

  [git cherry-pick]

  // е°ҶжҢҮе®ҡзҡ„жҸҗдәӨпјҲcommitпјүеә”з”ЁдәҺе…¶д»–еҲҶж”ҜгҖӮ
  git cherry-pick <commitHash>

   a - b - c - d   Master
         \
           e - f - g Feature

  зҺ°еңЁе°ҶжҸҗдәӨ f еә”з”ЁеҲ° master еҲҶж”ҜгҖӮпјҢеҲҮжҚўеҲ° master еҲҶж”Ҝ `git checkout master` з„¶еҗҺжү§иЎҢ `git cherry-pick f`

   a - b - c - d - f   Master
       \
         e - f - g Feature

е°ҸзҹҘиҜҶпјҡ
  1. git cherry-pick е‘Ҫд»Өзҡ„еҸӮж•°пјҢдёҚдёҖе®ҡжҳҜжҸҗдәӨзҡ„е“ҲеёҢеҖјпјҢеҲҶж”ҜеҗҚд№ҹжҳҜеҸҜд»Ҙзҡ„пјҢиЎЁзӨәиҪ¬з§»иҜҘеҲҶж”Ҝзҡ„жңҖж–°жҸҗдәӨ
     > git cherry-pick feature // featureеҲҶж”Ҝзҡ„жңҖиҝ‘дёҖж¬ЎжҸҗдәӨпјҢиҪ¬з§»еҲ°еҪ“еүҚеҲҶж”ҜгҖӮ

  2. еҰӮжһңйңҖиҰҒеҗҢж—¶ cherry-pickеӨҡдёӘ <commitHash>
    > git cherry-pick <commitHash1> <commitHash2>

  3. жғіиҰҒеңЁдёҚеҗҢеҲҶж”Ҝй—ҙ й«ҳж•Ҳзҡ„дҪҝз”Ё cherry-pickпјҢжҸҗдәӨд»Јз Ғзҡ„з»ҶзІ’еәҰеҝ…йЎ»иҰҒе°ҸпјҢдё”жҜҸж¬ЎжҸҗдәӨзҡ„еҠҹиғҪзӮ№е°ҪйҮҸеҚ•дёҖ
     еҰӮпјҡеҠҹиғҪзӮ№йғҪжҳҜеҸҜеӨҚз”Ёзҡ„дёҡеҠЎйҖ»иҫ‘пјҢжүҚиғҪжӣҙеҘҪзҡ„й«ҳж•ҲеӨҚз”ЁгҖӮ

  [git rebase]

  git stash // иҙ®и—Ҹжң¬ең°д»Јз Ғ

  git pull origin latest --rebase // жӢүеҸ–иҝңзЁӢlatestеҲҶж”Ҝд»Јз Ғ

  git stash pop // иҝҳеҺҹжң¬ең°иҙ®и—Ҹд»Јз Ғ
